# 🏥 DietMed Global - 다이어트 의약품 검증 플랫폼

## 📋 플랫폼 소개

**DietMed Global**은 전 세계 다이어트 의약품의 안전성을 검증하고 비교하는 글로벌 정보 플랫폼입니다.

### 🎯 핵심 가치

> **"살 빼는 정보, 이제는 검증부터"**

- **검증 우선**: 광고나 SNS 후기가 아닌 국가 허가 기준
- **안전 중심**: 빠른 감량보다 안전한 선택
- **신뢰 정보**: 정부 허가·성분·부작용 투명 공개

### 🌟 차별화 포인트

| 영역 | 일반 플랫폼 | DietMed Global |
|------|------------|----------------|
| **정보 출처** | SNS 후기, 광고 | 정부 기관 공식 데이터 |
| **검증 기준** | 자체 기준 | FDA·MFDS·EMA 국제 표준 |
| **안전 점수** | 없음 | 0-100점 4단계 등급 |
| **비교 기능** | 단순 나열 | 28개 항목 상세 비교 |
| **금지 성분** | 일부만 | 블랙리스트 완전 공개 |
| **다국어** | 1-2개 | 9개 국어 지원 |

## ✅ 완료된 핵심 기능

### 1️⃣ 제품 검증 시스템

#### 📱 바코드 스캔
- **즉시 검증**: 바코드 입력으로 즉시 제품 확인
- **국가별 허가 상태**: 30개 글로벌 규제기관 정보
- **위험도 분석**: 금지 성분, 냉장 유통 요구사항 체크
- **승인 국가 수**: 제품이 승인된 국가 개수 표시

**테스트 바코드**: `0169-4517-02` (Wegovy)

#### 🔍 제품 검색
- **실시간 검색**: 제품명 또는 성분명으로 검색
- **국가별 필터링**: 특정 국가의 승인 제품만 조회
- **리스크 배지**: 안전/주의/위험 시각적 표시
- **승인 개수**: 각 제품의 글로벌 승인 현황

### 2️⃣ 안전성 평가 시스템

#### 📊 소비자 안전 점수 (0-100점)

**점수 구성** (4개 평가 영역):
```
① 허가/정품성 (35점)
   - 정부 기관 공식 승인 여부
   - 승인 번호 및 제조사 검증
   - 라벨·시리얼·바코드 진위 확인

② 근거/효과 (25점)
   - 임상시험(RCT) 근거
   - 6개월/12개월 체중 감량 데이터
   - 중단 후 리바운드 정보

③ 안전성/부작용 (25점)
   - 주요 부작용 공개
   - 금기사항 명시
   - 리콜·경고 이력 관리

④ 유통/추적성 (15점)
   - 병원/약국 공식 유통
   - 판매자 정보 투명성
```

**안전 등급**:
- 🟢 **90-100점**: 매우 안전 (정부 승인, 강력한 임상 근거)
- 🟡 **70-89점**: 비교적 안전 (승인 확인, 일부 정보 부족)
- 🟠 **50-69점**: 주의 필요 (승인 불명확, 임상 근거 약함)
- 🔴 **0-49점**: 고위험 (금지 성분 또는 미승인)

**현재 데이터**:
- Wegovy (Semaglutide): 97점 (Green)
- Saxenda (Liraglutide): 95점 (Green)
- Contrave: 85점 (Light Green)
- Qsymia: 83점 (Light Green)

#### 🛡️ 안전 프로필

각 성분별 상세 정보:
- **효능**: 6/12개월 평균 체중 감량률
- **작용 기전**: 약물의 작용 원리
- **흔한 부작용**: 발생률과 함께 표시
- **심각한 부작용**: 주의해야 할 중대 부작용
- **금기사항**: 복용하면 안 되는 경우
- **약물 상호작용**: 병용 시 주의할 약물
- **임신/수유**: 안전성 등급
- **중독 위험도**: Low/Moderate/High

### 3️⃣ 제품 비교 시스템

#### ⚖️ 다중 제품 비교 (2-4개)

**비교 항목 (28개)**:

**기본 정보** (5개)
- 제품명, 성분명, 제조사, 제형, 투여 경로

**허가 정보** (6개)
- 허가 국가 수, 승인 번호, 승인 일자
- 처방 구분 (Rx/OTC), BMI 기준, 연령 제한

**효능** (5개)
- 작용 기전
- 6개월/12개월 감량률
- 효과 발현 시기
- 임상 근거 수준

**안전성** (6개)
- 흔한/심각한 부작용
- 금기사항
- 약물 상호작용
- 임신/수유 안전성
- 중독 위험도

**안전 점수** (6개)
- 총점 (0-100)
- 허가/효능/안전/유통 세부 점수
- 안전 등급 (Green/Yellow/Red)

**사용 방법**:
1. 검색 결과에서 제품 체크박스 선택 (2-4개)
2. 우측 하단 "비교하기" 버튼 클릭
3. 28개 항목 상세 비교 테이블 표시
